What is JavaScript?

JavaScript is the programming language of the web. If HTML is the skeleton and CSS is the skin, JavaScript is the brain and muscles!

JavaScript can:

  • React to button clicks
  • Change content dynamically
  • Validate forms
  • Create animations
  • Build entire games!

Variables - Storing Information

Variables are like labeled boxes where you store data:

// Creating variables
let playerName = "Alex";        // Text (string)
let score = 0;                  // Number
let isPlaying = true;           // Boolean (true/false)

// You can change them later
score = 10;
playerName = "Sam";
Variable Rules:
  • Use let for values that change
  • Use const for values that stay the same
  • Names can't have spaces (use camelCase: myVariableName)
  • Names can't start with numbers

Try it yourself!

// Math with variables
let a = 5;
let b = 3;
let sum = a + b;        // sum is 8
let product = a * b;    // product is 15

// Text (string) operations
let firstName = "John";
let lastName = "Doe";
let fullName = firstName + " " + lastName;  // "John Doe"

Functions - Reusable Code

Functions are like recipes - a set of instructions you can use over and over:

// Creating a function
function greetPlayer(name) {
    alert("Welcome, " + name + "!");
}

// Using (calling) the function
greetPlayer("Alice");   // Shows: Welcome, Alice!
greetPlayer("Bob");     // Shows: Welcome, Bob!

Functions can return values:

function addNumbers(x, y) {
    return x + y;
}

let result = addNumbers(5, 3);  // result is 8
Common Mistake: Forgetting the parentheses () when calling a function. Always write myFunction(), not just myFunction

If/Else - Making Decisions

Programs need to make decisions based on conditions:

let age = 18;

if (age >= 18) {
    console.log("You can vote!");
} else {
    console.log("Too young to vote.");
}

// Multiple conditions
let score = 85;

if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else if (score >= 70) {
    console.log("Grade: C");
} else {
    console.log("Grade: F");
}
Comparison Operators:
  • === equal to
  • !== not equal to
  • > greater than
  • < less than
  • >= greater than or equal
  • <= less than or equal

üéÆ Let's Build: Number Guessing Game!

Now let's use everything we learned to build an interactive game!

The Complete Game

Create a new file called number-game.html:

<!DOCTYPE html>
<html>
<head>
    <title>Number Guessing Game</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            max-width: 500px;
            margin: 50px auto;
            text-align: center;
            background-color: #f0f8ff;
        }
        
        .game-container {
            background: white;
            padding: 30px;
            border-radius: 15px;
            box-shadow: 0 5px 15px rgba(0,0,0,0.1);
        }
        
        input {
            font-size: 20px;
            padding: 10px;
            width: 100px;
            margin: 10px;
            border: 2px solid #007bff;
            border-radius: 5px;
        }
        
        button {
            font-size: 18px;
            padding: 10px 20px;
            margin: 10px;
            background-color: #007bff;
            color: white;
            border: none;
            border-radius: 5px;
            cursor: pointer;
        }
        
        button:hover {
            background-color: #0056b3;
        }
        
        #message {
            font-size: 20px;
            margin: 20px;
            height: 30px;
        }
        
        .win { color: #28a745; }
        .high { color: #dc3545; }
        .low { color: #ffc107; }
    </style>
</head>
<body>
    <div class="game-container">
        <h1>üé≤ Number Guessing Game üé≤</h1>
        <p>I'm thinking of a number between 1 and 100!</p>
        
        <input type="number" id="guessInput" min="1" max="100" placeholder="?">
        <br>
        <button onclick="checkGuess()">Guess!</button>
        <button onclick="newGame()">New Game</button>
        
        <div id="message"></div>
        <p>Attempts: <span id="attempts">0</span></p>
    </div>

    <script>
        // Game variables
        let secretNumber = Math.floor(Math.random() * 100) + 1;
        let attempts = 0;

        // Function to check the player's guess
        function checkGuess() {
            // Get the player's guess
            let guess = document.getElementById('guessInput').value;
            let message = document.getElementById('message');
            
            // Increase attempts
            attempts++;
            document.getElementById('attempts').textContent = attempts;
            
            // Convert guess to number
            guess = parseInt(guess);
            
            // Check if guess is valid
            if (isNaN(guess) || guess < 1 || guess > 100) {
                message.textContent = "Please enter a number between 1 and 100!";
                message.className = "";
                return;
            }
            
            // Check the guess
            if (guess === secretNumber) {
                message.textContent = "üéâ Congratulations! You got it in " + attempts + " tries!";
                message.className = "win";
            } else if (guess > secretNumber) {
                message.textContent = "Too high! Try a lower number.";
                message.className = "high";
            } else {
                message.textContent = "Too low! Try a higher number.";
                message.className = "low";
            }
        }

        // Function to start a new game
        function newGame() {
            secretNumber = Math.floor(Math.random() * 100) + 1;
            attempts = 0;
            document.getElementById('attempts').textContent = attempts;
            document.getElementById('message').textContent = "";
            document.getElementById('guessInput').value = "";
        }

        // Allow Enter key to submit guess
        document.getElementById('guessInput').addEventListener('keypress', function(event) {
            if (event.key === 'Enter') {
                checkGuess();
            }
        });
    </script>
</body>
</html>

How the game works:

  1. Math.random() generates a random decimal between 0 and 1
  2. We multiply by 100 and add 1 to get 1-100
  3. Math.floor() rounds down to whole number
  4. We track attempts and give hints (too high/low)
  5. The new game button resets everything
