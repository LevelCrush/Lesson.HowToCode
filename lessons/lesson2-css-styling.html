What is CSS?

CSS stands for Cascading Style Sheets. If HTML is the skeleton, CSS is the skin, clothes, and makeup!

See the difference!

Without CSS:

Hello World

With CSS:

Hello World

CSS Syntax

CSS has a simple pattern: select something, then style it!

selector {
    property: value;
    another-property: another-value;
}

Real Example:

h1 {
    color: blue;
    font-size: 36px;
    text-align: center;
}

Styled Heading!

Three ways to add CSS:
  1. Inline: <p style="color: red;">Red text</p>
  2. Internal: <style> p { color: red; } </style> in the head
  3. External: <link rel="stylesheet" href="styles.css"> (Best way!)

Colors & Fonts

The Box Model

Every element is a box with four parts:

Content
Padding
Border
Margin 
div {
    margin: 20px;      /* Space outside */
    border: 2px solid black;  /* The border */
    padding: 15px;     /* Space inside */
    width: 200px;      /* Content width */
}

üéÆ Let's Style: Make Your Quiz Beautiful!

Now let's add CSS to transform your quiz game!

Step 1: Create a CSS file

Create a new file called quiz-styles.css with this code:

/* General Styles */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f8ff;
    margin: 0;
    padding: 20px;
}

#game-container {
    max-width: 600px;
    margin: 0 auto;
    background-color: white;
    padding: 30px;
    border-radius: 15px;
    box-shadow: 0 5px 15px rgba(0,0,0,0.1);
}

/* Title Styles */
h1 {
    color: #333;
    text-align: center;
    margin-bottom: 30px;
}

/* Question Box */
#question-box {
    background-color: #e7f3ff;
    padding: 20px;
    border-radius: 10px;
    margin-bottom: 20px;
}

#question-number {
    color: #007bff;
    font-size: 20px;
}

#question-text {
    font-size: 18px;
    color: #333;
    margin: 15px 0;
}

/* Answer Buttons */
#answer-buttons {
    display: grid;
    gap: 10px;
}

button {
    background-color: #007bff;
    color: white;
    border: none;
    padding: 15px 20px;
    font-size: 16px;
    border-radius: 8px;
    cursor: pointer;
    transition: background-color 0.3s;
}

button:hover {
    background-color: #0056b3;
}

/* Score Section */
#score-section {
    text-align: center;
    font-size: 20px;
    color: #28a745;
    font-weight: bold;
}

Step 2: Link CSS to your HTML

Update your quiz HTML file to include the CSS. Add this line in the <head> section:

<head>
    <title>My Awesome Quiz Game</title>
    <link rel="stylesheet" href="quiz-styles.css">
</head>

<!DOCTYPE html>
