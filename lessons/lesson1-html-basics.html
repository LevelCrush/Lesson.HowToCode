What is HTML?

HTML stands for HyperText Markup Language. Think of it as the blueprint of a house - it defines the structure and content of your webpage.

Try this!

HTML uses tags to mark up content. Tags are like containers:

<p>This is a paragraph!</p>

The <p> is the opening tag, and </p> is the closing tag.

Basic HTML Structure

Every HTML page follows this basic structure:

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first webpage.</p>
</body>
</html>
What each part does:
  • <!DOCTYPE html> - Tells the browser this is HTML5
  • <html> - The root element that contains everything
  • <head> - Contains metadata (info about the page)
  • <title> - Sets the browser tab title
  • <body> - Contains all visible content

Common HTML Tags

Here are the most useful HTML tags you'll need:

Text Tags

  • <h1> to <h6> - Headings (h1 is biggest)
  • <p> - Paragraph
  • <strong> - Bold text
  • <em> - Italic text

Container Tags

  • <div> - Generic container
  • <button> - Clickable button
  • <input> - Input field
  • <form> - Form container

Examples in action:

<h1>Big Title</h1>
<p>This is a <strong>bold</strong> word.</p>
<button>Click me!</button>

üéÆ Let's Build: Quiz Game Structure!

Now let's use what we learned to build the HTML structure for a quiz game!

Step 1: Create the basic structure

Copy this code and save it as my-quiz-game.html:

<!DOCTYPE html>
<html>
<head>
    <title>My Awesome Quiz Game</title>
</head>
<body>
    <h1>Welcome to the Quiz Game!</h1>
    
    <div>
        <h2>Question 1:</h2>
        <p>What does HTML stand for?</p>
        
        <button>A. Hyper Text Markup Language</button>
        <button>B. Hot Tamale Making Language</button>
        <button>C. How To Make Lasagna</button>
        <button>D. Heavy Tool Moving Language</button>
    </div>
    
    <div>
        <p>Score: 0</p>
    </div>
</body>
</html>

Step 2: Add more structure

Let's improve our quiz by adding IDs (unique names) to elements:

<!DOCTYPE html>
<html>
<head>
    <title>My Awesome Quiz Game</title>
</head>
<body>
    <div id="game-container">
        <h1>Welcome to the Quiz Game!</h1>
        
        <div id="question-box">
            <h2 id="question-number">Question 1:</h2>
            <p id="question-text">What does HTML stand for?</p>
            
            <div id="answer-buttons">
                <button id="btn-a">A. Hyper Text Markup Language</button>
                <button id="btn-b">B. Hot Tamale Making Language</button>
                <button id="btn-c">C. How To Make Lasagna</button>
                <button id="btn-d">D. Heavy Tool Moving Language</button>
            </div>
        </div>
        
        <div id="score-section">
            <p>Score: <span id="score">0</span></p>
        </div>
    </div>
</body>
</html>
Why use IDs?

IDs are like name tags for your HTML elements. Later, we'll use CSS to style specific elements and JavaScript to make them interactive!
